# Babo AI Assistant

A personal AI desktop assistant built in Python.

## Features

- Voice Activation
- Speech Recognition
- Text-to-Speech
- Application Control
- Browser Automation
- Folder Navigation
- Media Controls
- Windows Automation
- Developer Automation

## Technologies

- Python
- Faster Whisper
- pyttsx3
- PyAutoGUI
- PyCAW
- psutil
